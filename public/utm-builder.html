UTM 빌더

채널별 유입을 추적할 UTM 링크를 만들고 관리합니다

UTM이 뭔가요? (처음이라면 꼭 읽어보세요!)

▼
UTM(Urchin Tracking Module)은 URL 뒤에 붙이는 추적 코드예요.
고객이 어디서, 어떤 경로로, 어떤 캠페인을 통해 우리 사이트에 들어왔는지 구글 애널리틱스에서 확인할 수 있게 해줍니다.

예를 들어, 같은 상품 페이지 링크를 네이버 블로그에도, 인스타그램에도 올렸을 때
UTM 없이는 "어디서 들어온 고객이 더 많이 구매했는지" 알 수 없어요.
UTM을 붙이면 채널별 성과를 정확하게 비교할 수 있습니다!
실제 UTM URL 예시
https://seamonster.co.kr/product/shrimp?
  utm_source=naver& utm_medium=blog& utm_campaign=2026_spring_sale
→ 이 URL을 클릭한 사람은 GA에서 "네이버 블로그의 2026 봄 세일 캠페인"에서 유입된 것으로 기록됩니다.
각 파라미터가 하는 역할
필수
utm_source
어디서? — 고객이 유입된 플랫폼/사이트
naver, google, kakao, instagram
필수
utm_medium
어떤 방식으로? — 광고/콘텐츠의 유형
cpc(클릭광고), blog, social, email
선택 (강력 권장)
utm_campaign
어떤 캠페인? — 진행 중인 프로모션/이벤트명
2026_spring_sale, new_product
선택
utm_content
어떤 소재? — 같은 캠페인 내 A/B 소재 구분
banner_a, video_15s, text_link
선택
utm_term
어떤 키워드? — 검색광고에서 사용한 키워드
해산물, 새우, 씨몬스터
작성 팁
  • 소문자로 통일하세요 — GA에서 Naver와 naver는 다른 소스로 집계됩니다
  • 띄어쓰기 대신 언더바(_)를 사용하세요 — 예: spring sale → spring_sale
  • 날짜나 시즌을 앞에 붙이면 나중에 찾기 쉬워요 — 예: 2026_spring_sale
  • 팀 내에서 네이밍 규칙을 통일하면 데이터 분석이 훨씬 수월합니다
UTM 파라미터 설정
고객이 최종적으로 도착할 페이지 주소예요.
상품 페이지, 이벤트 페이지, 메인 페이지 등 링크를 공유할 목적지 URL을 넣으세요.
고객이 어디에서 이 링크를 클릭하나요?
링크를 올릴 플랫폼을 선택하세요. 예) 네이버 블로그에 올릴 거면 → naver
어떤 방식으로 링크를 노출하나요?
클릭당 과금 광고면 → cpc / 블로그 포스팅이면 → blog / SNS 게시물이면 → social
이 링크가 속한 캠페인(프로모션/이벤트) 이름이에요.
나중에 GA에서 캠페인별 성과를 비교할 때 사용됩니다. 팀원들이 알아볼 수 있는 이름으로!
같은 캠페인에서 여러 광고 소재를 테스트할 때 구분용으로 사용해요.
예) 배너 A vs 배너 B, 15초 영상 vs 30초 영상 → 어떤 소재가 더 효과적인지 비교 가능!
주로 검색 광고에서 사용해요. 어떤 검색어에 광고를 노출했는지 기록합니다.
검색 광고가 아니면 비워두셔도 됩니다.
이 URL에 대한 내부 메모를 남겨두세요.
GA에는 전송되지 않고, 히스토리에서만 확인할 수 있습니다.

양식으로 대량 추가

양식 다운로드 후 엑셀에서 한 줄에 하나씩 채우고 파일 업로드 하면 UTM URL이 자동 생성돼 히스토리에 추가됩니다.
열: 기본URL · 소스 · 미디움 · 캠페인 · 콘텐트 · 텀 · 비고 (기본URL·소스·미디움 필수 / 예시 행은 지우고 입력)
